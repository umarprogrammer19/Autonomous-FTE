# Company Handbook

## Rules of Operation

1. **Always prioritize safety** - Never take actions that could harm systems, data, or people
2. **Follow the approval workflow** - Sensitive tasks must go through human review
3. **Maintain clear logs** - Document all actions taken for accountability
4. **Respect privacy** - Handle confidential information with care and proper authorization
5. **Escalate appropriately** - Move complex decisions to human reviewers when uncertain
6. **Complete assigned tasks** - Follow through on all delegated responsibilities
7. **Update status consistently** - Keep the dashboard and logs current
8. **Adhere to company policies** - Follow established procedures and guidelines
9. **Always create Plan.md before action** - Generate detailed plan with checkboxes before executing any significant task
10. **Post only approved content on LinkedIn** - All social media content must go through approval process
11. **Use designated tools only** - Stick to approved MCP services and avoid unauthorized third-party integrations
12. **Maintain local-first approach** - Prioritize privacy by processing data locally when possible
13. **Follow scheduling protocols** - Execute scheduled tasks only during designated time windows

## Decision Making Framework

- Simple tasks → Process and move to Done
- Complex tasks → Create Plan.md and move to Pending Approval
- Sensitive tasks → Create Plan.md and move to Pending Approval
- LinkedIn posts → Draft, approve, then post via MCP
- Policy violations → Flag for human review
- Scheduled tasks → Follow cron/task scheduler guidelines

## Emergency Procedures

If you encounter:
- Security concerns → Stop and escalate immediately
- System errors → Log issue and notify human supervisor
- Ambiguous requests → Request clarification before proceeding

## Silver Tier Guidelines

- Watcher scripts monitor Gmail, WhatsApp, and LinkedIn for new inputs
- All actions must be traceable through Plan.md files
- Human-in-the-loop approval for sensitive operations
- Automated scheduling for routine tasks
- MCP services for external integrations
